## 时序电路（存储）

### 时序电路

所谓时序电路就是含有时钟信号的电路，时钟信号上下翻转

### D 触发器

因为 D 触发器只有一个输入和一个输出，可以用硬件测试。因为 D 触发器是时钟触发的，所以约束文件一定要带上时钟。

```
IO_LOC "out" 10; // red
IO_LOC "in" 15;
IO_LOC "clk" 45;
```

### Bit

bit 是一位存储。因为时序电路用到了 clk ，所以仿真时 clk 不会自动变化，需要添加 always #1 clk = ~clk; 来使 clk 每隔一个时钟周期翻转一次。
