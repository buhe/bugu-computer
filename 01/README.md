## 布尔逻辑
大部分电路输入都大于两个，如 Mux 有三个输入，And16 甚至有 16 个输入。只有 DMux 有两个输入两个输出。

虽然 tangnaono 只有一个 led 能被程序控制，但 tangnano 有很多 gpio ，gpio 是通用接口，程序可以控制它产生高低电平（也就是 0 1 ）。可以把一个 led 接在 gpio 上（这里接到 2）这样控制 gpio 的电平可以间接控制 led 的灯亮灭。
